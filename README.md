This is Full-CI code I wrote to further learn the language of second-order quantization, following the Full-CI instructions from Garnet Chan's group (https://github.com/gkclab/pickle).

The paper in the paper_instructions section is about the kh-fci algorithm. Then there's an FCI-instructions document which is Garnet Chan's FCI tutorial. The two folders (H6*) are preliminary implementations of this tutorial. The kh-fci and sc-fci folders are self-explanatory. funs.py contains the functions used to implement the corresponding algorithms, and main.py provides an example, externally using pyscf to provide Hartree-Fock and integration.
